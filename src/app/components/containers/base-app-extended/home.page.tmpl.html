What is Spyne?

Spyne is a next-gen Javascript Framework designed from the ground up to efficiently develop the most demanding single-page-applications.
The framework combines traditional coding principles with modern patterns to create the cleanest, most efficient Javascript codebase.

How to Generate Your Single-Page-App

After customizing the menu, your custom application outputs a data-file, html-templates and view-components (ViewStreams) that can be further customized with Javascript, Sass and Javascript libraries.
